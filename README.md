# go web socket

#### 项目介绍
websocket   go作为后台使用

#### 软件架构
软件架构说明


#### 安装教程

1. 直接运行
2. 服务器需要修改内网ip
3. 需要端口 1234 可以修改

#### 使用说明

1. 本地运行就ok 如果不行查看1234端口是否被占用
2. 服务器修改成内网Ip 安全组 开放1234端口 
3. 服务器可能需要文件权限 chmod+x file... 然后才能执行 ./file...
